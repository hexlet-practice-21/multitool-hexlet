Это тестовое приложение для колледжа, которое предоставляет возможность конвертировать 
различные единицы измерения, такие как длина, скорость, вес и объем.
### Технологии:
- HTML
- CSS
- JavaScript
- Bootstrap

### Установка:
1) git clone https://github.com/hexlet-practice-21/multitool-hexlet
2) make install
3) make build
4) make run
